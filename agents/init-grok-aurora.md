## Design Guidelines

### DO:
- Use the brand colour palette
- Create clean, minimal compositions
- Design for dark backgrounds (navy) primarily
- Include negative space
- Think about scalability
- Create abstract/geometric visuals over literal representations

### DON'T:
- Use stock photo aesthetics
- Create cluttered compositions
- Use colours outside the brand palette
- Add unnecessary decoration
- Create assets that won't scale well
